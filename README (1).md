# Registro de Actividades — Neonatología HSVP

App personal para registrar diariamente los procedimientos y actividades
realizadas durante el turno en la UCIN, con exportación a Excel para el
informe semestral de actividades.

- Los datos se guardan únicamente en el dispositivo (localStorage).
- Usa el botón "Descargar respaldo (.json)" en Ajustes para respaldar
  periódicamente y subirlo a Drive u otro almacenamiento propio.
- Autor: Dr. Carlos Culki — uso personal.
